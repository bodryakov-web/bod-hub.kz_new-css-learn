# Описание проекта NewCSSLearn

## Общее описание

NewCSSLearn - это full-stack web-приложение для обучения студентов новейшим свойствам CSS. Приложение представляет собой образовательную платформу с последовательностью уроков, разбитых на разделы, с возможностью тестирования и выполнения практических задач.

## Архитектурные особенности

### Backend
- **Чистый PHP 8.2** без фреймворков и библиотек
- **MySQL база данных** с хранением контента в формате JSON
- **Собственный роутинг** через .htaccess и PHP
- **CRUD операции** для управления разделами и уроками
- **Загрузка изображений** в файловую систему

### Frontend
- **Чистый HTML5** с семантической разметкой
- **Современный CSS** с Grid layout и Material Design
- **Ванильный JavaScript** без библиотек
- **Адаптивный дизайн** с breakpoint'ами от 320px до 1920px+
- **Темная/светлая тема** с переключением

### База данных
- **Две таблицы**: sections и lessons
- **UTF-8 mb4** кодировка для корректной работы с кириллицей
- **JSON хранение** контента уроков
- **Связи** один-ко-многим между разделами и уроками

## Функциональные возможности

### Для пользователей
1. **Просмотр разделов** в виде карточек на главной странице
2. **Навигация по урокам** внутри разделов
3. **Чтение теоретического материала** с форматированным текстом
4. **Прохождение тестов** с мгновенной обратной связью
5. **Просмотр задач** с условиями и примерами кода
6. **Хлебные крошки** для навигации

### Для администратора
1. **Авторизация** по логину/паролю
2. **Управление разделами** (создание, редактирование, удаление)
3. **Управление уроками** (создание, редактирование, удаление)
4. **Редактор Quill** для форматирования текста
5. **Загрузка изображений** до 5МБ
6. **Управление тестами** (добавление/удаление вопросов)
7. **Публикация/черновики** уроков

## Структура URL

- **Главная страница**: `/` - отображение разделов
- **Страница урока**: `/section_number-{section_slug}/lesson_number-{lesson_slug}`
- **Админ-панель**: `/bod` - вход в админку
- **404 страница**: любой некорректный URL

## Дизайн и UX

### Material Design принципы
- **Плоский дизайн** с тенями и глубиной
- **Цветовая схема**: серые, голубые, синие оттенки
- **Шрифт Roboto** из Google Fonts
- **Grid layout** вместо Flexbox
- **Плавная прокрутка** и анимации

### Адаптивность
- **Мобильные**: 320px - 599px
- **Малые устройства**: 600px - 899px  
- **Планшеты**: 900px - 1199px
- **Ноутбуки**: 1200px - 1599px
- **Большие мониторы**: 1600px - 1919px
- **Сверхширокие**: 1920px+

### Темы оформления
- **Светлая тема** (по умолчанию)
- **Темная тема** с переключателем в header
- **Сохранение выбора** в localStorage

## Технические особенности

### Безопасность
- **Базовая защита** от XSS через htmlspecialchars
- **Валидация** slug'ов и обязательных полей
- **Авторизация** админ-панели

### Производительность
- **Кэширование** статических ресурсов
- **Оптимизация изображений** (ограничение размера)
- **Эффективный роутинг** без лишних запросов

### SEO оптимизация
- **Семантическая HTML5 разметка**
- **Микроразметка** для образовательного контента
- **Человекопонятные URL** с slug'ами

## Docker конфигурация

### Локальная разработка
- **PHP 8.2 + Apache** контейнер
- **MySQL 8.0** база данных
- **Volume монтирование** для разработки
- **Автоматическая инициализация** БД

### Продакшн готовность
- **Простой деплой** на любой хостинг с PHP

## Тестовые данные

Приложение включает:
- **2 раздела** по 2 урока в каждом для демонстрации
- **Примеры тестов** с 3 вопросами в первом уроке
- **Образцы контента** с изображениями и форматированием
