# Последовательность создания проекта NewCSSLearn

## Этап 1: Инфраструктура и база данных

### 1.1 Создание Docker конфигурации
- Создать Dockerfile с PHP 8.2 + Apache + расширения MySQL
- Создать docker-compose.yml с PHP и MySQL сервисами
- Настроить volumes для монтирования кода и данных MySQL

### 1.2 Создание SQL скриптов
- db-init.sql: создание таблиц sections и lessons
- db-test-data.sql: заполнение тестовыми данными (2 раздела, 4 урока)
- Установка правильной кодировки utf8mb4_unicode_ci

### 1.3 Создание базовой файловой структуры
- Создать все директории согласно structure.md
- Создать пустые файлы с правильной кодировкой UTF-8 без BOM

## Этап 2: Backend ядро

### 2.1 Конфигурация и подключение к БД
- config.php: параметры подключения к MySQL
- Database.php: класс для работы с базой данных
- functions.php: вспомогательные функции

### 2.2 Роутинг и MVC архитектура
- .htaccess: настройка URL rewriting
- index.php: главный файл с инициализацией и роутером
- Router.php: класс обработки URL маршрутов

### 2.3 Базовые модели и логика
- Создание базовых CRUD операций для sections и lessons
- Валидация данных (slug, обязательные поля)
- Обработка JSON контента уроков

## Этап 3: Frontend основа

### 3.1 Базовые шаблоны
- templates/header.php: шапка сайта с навигацией и темой
- templates/footer.php: подвал сайта
- templates/404.php: страница ошибки

### 3.2 CSS архитектура
- assets/css/main.css: основные стили с Material Design
- assets/css/themes.css: светлая и темная темы
- Настройка адаптивности с указанными breakpoint'ами
- Подключение шрифта Roboto из Google Fonts

### 3.3 JavaScript функциональность
- assets/js/theme-toggle.js: переключение тем
- assets/js/main.js: базовая интерактивность
- Работа с data-атрибутами вместо классов/ID

## Этап 4: Основные страницы

### 4.1 Главная страница с разделами
- templates/sections.php: карточки разделов
- Отображение только опубликованных уроков
- Ссылки на уроки с правильной структурой URL

### 4.2 Страница урока
- templates/lesson.php: полный шаблон урока
- Динамическая генерация из JSON данных
- Хлебные крошки
- Кнопки навигации (предыдущий/следующий/оглавление)

### 4.3 Система тестов
- assets/js/quiz.js: логика тестов с мгновенной обратной связью
- Подсветка правильных/неправильных ответов
- Генерация тестов из JSON данных

## Этап 5: Админ-панель

### 5.1 Система авторизации
- admin/login.php: форма входа
- admin/logout.php: выход
- Проверка авторизации для всех админ страниц
- Хранение сессии администратора

### 5.2 Интерфейс управления разделами
- admin/sections.php: список разделов с CRUD операциями
- admin/edit-section.php: создание/редактирование раздела
- Валидация уникальности slug и порядка

### 5.3 Интерфейс управления уроками
- admin/lessons.php: список уроков с фильтрацией по разделам
- admin/edit-lesson.php: создание/редактирование урока
- Интеграция редактора Quill

### 5.4 Редактор контента
- Подключение Quill.js через CDN
- admin/templates/lesson-form.php: форма с редактором
- Сохранение контента в формате JSON
- Управление тестами (добавление/удаление вопросов)

## Этап 6: Работа с изображениями

### 6.1 Загрузка изображений
- admin/upload.php: обработка загрузки файлов
- Валидация форматов (png, jpg, webp, svg)
- Ограничение размера до 5МБ
- Создание директорий уроков

### 6.2 Управление изображениями

- Отображение в тексте уроков

## Возможные проблемы и решения

### Проблема: Кодировка UTF-8
**Решение**: Проверить все файлы на отсутствие BOM, установить правильные заголовки

### Проблема: Права доступа к изображениям
**Решение**: Установить правильные права на папку uploads

### Проблема: SQL ошибки
**Решение**: Проверить кодировку таблиц и соединения с БД

### Проблема: URL роутинг не работает
**Решение**: Проверить .htaccess и настройки Apache

### Проблема: Quill редактор не загружается
**Решение**: Проверить CDN подключения и порядок загрузки скриптов
